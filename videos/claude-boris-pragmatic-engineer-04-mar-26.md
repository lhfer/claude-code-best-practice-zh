## 这期最值得听什么

这期更偏工程视角。
如果你关心的不是“怎么用”，而是“为什么这个产品会长成这样”，这期很值。

## 5 条高信号 takeaway

1. **Claude Code 的设计不是围绕单个 prompt，而是围绕整个工程流程**
2. **并行、多 worktree、多 agent 是核心生产力假设**
3. **产品要对未来模型能力做提前设计**
4. **很多 today-best-practice 其实会随着模型进化继续重写**
5. **真正关键的是上下文、工具接入和验证，而不只是代码生成本身**
